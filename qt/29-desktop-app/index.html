different between desktop application and mobile app

desktop application

  • you can use Qt Widgets or QML
  • you can use any type of databases
  • you can use ui framework, for example: qml-bootstrap
  • you can access database directly between your application and server

mobile app

  • you'd better use QML
  • you can use sqlite on your mobile device
  • you can use ui framework too, for example: qml-bootstrap
  • you cann't access remote database directly from your mobile device, but you can use rest/serving ui via http/templating/http request/local files/web sockets, etc. For example:
https://github.com/qtrest/qtrest
https://github.com/Skycoder42/QtRestClient
