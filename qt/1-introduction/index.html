QTCreator

how to

  • New Project
"Application -> Qt Widgets Application" for general applications on desktop
"Non-Qt Project -> Plain C++ Application" for general C++ console applications

  • Color in Editor: In Tools -> Options -> Color Scheme for Theme ...

  • Vim mode: In Tools -> Options -> FakeVim

some shutcuts

  • build: Ctl+B
  • run: Ctl+R
  • breakpoint: F9
  • fullscreen: Ctl+Shift+F11
  • help: Ctl+6, Alt+L to "Look for", Esc to exit help
  • sub-wind in foot: Alt+num
  • other source file: Alt+arrow (left & right)
  • left sidebar: Alt+0
  • right sidebar: Alt+Shift+0
  • preview ui: alt+shift+r(Tools->Options->Environment->Keyboard->FormEditor)

some skills

  • layout: Ctl+H to H, Ctl+L to V
  • source/header: F4
  • layout中可以加入弹簧把其他组件顶起来: layout->addStretch()
  • 组件中可以发射自身的信号, 相当于调用这个信号对应的所有槽函数, emit findPrevious(text, cs);
  • QWidget/QDialog/QMainWindow的layout是不同的, QWidget/QDialog中可以用setLayout, QMainWindow中用setCentralWidget添加一个Widget, 因为QMainWindow之中已经有了layout
  • 组件显示字符串中&C是快捷键
  • in Forms->ui editor, you can add spacer, F3 to widget editor, F4 to signal/slot editor
  • fixed window size: max & min size to be the same
  • qDebug(#include <QtDebug>) to output debug logs
